## Homebrew without xcode
### Install Command Line Tools

```
xcode-select --install  
```

When this is finished, you can install homebrew!

### Install Homebrew
```
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
Voila
